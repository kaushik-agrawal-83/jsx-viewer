## [1.0.0](https://github.com/kaushikagrawal/jsx-viewer/releases/tag/v1.0.0) (2026-06-08)

### Features

* **native app**: Tauri v2 migration — native macOS desktop app with full filesystem access and OS drag-and-drop ([f8f3b2a](https://github.com/kaushikagrawal/jsx-viewer/commit/f8f3b2a))
* **file associations**: `.jsx` and `.tsx` files appear in Finder "Open With"; handles cold and warm launches via `RunEvent::Opened` ([580a511](https://github.com/kaushikagrawal/jsx-viewer/commit/580a511))
* **default handler prompt**: first-launch modal to set JSX Viewer as default for `.jsx`/`.tsx` files ([580a511](https://github.com/kaushikagrawal/jsx-viewer/commit/580a511))
* **split pane**: view two components side by side with a resizable divider ([f987917](https://github.com/kaushikagrawal/jsx-viewer/commit/f987917))
* **tabs**: multi-tab support per pane with session persistence across restarts ([f987917](https://github.com/kaushikagrawal/jsx-viewer/commit/f987917))
* **sidebar**: collapsible file sidebar with drag-and-drop, resize, `⌘B` toggle ([f987917](https://github.com/kaushikagrawal/jsx-viewer/commit/f987917), [d888c8b](https://github.com/kaushikagrawal/jsx-viewer/commit/d888c8b))
* **file watcher**: live reload on save; missing-file detection emits `file-deleted` event ([580a511](https://github.com/kaushikagrawal/jsx-viewer/commit/580a511))
* **pre-bundled libraries**: React 18, Tailwind CSS, Recharts, Lucide, Radix slot — no install required ([f987917](https://github.com/kaushikagrawal/jsx-viewer/commit/f987917))
* **web dev mode**: Docker + Express backend for development without Rust toolchain ([6012559](https://github.com/kaushikagrawal/jsx-viewer/commit/6012559))
