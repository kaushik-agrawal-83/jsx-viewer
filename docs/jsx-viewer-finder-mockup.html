Open .jsx files with JSX Viewer?
Double-clicking a .jsx file in Finder will open it directly in JSX Viewer.
Note: This only affects how .jsx files open — your code editor remains unchanged. You can reverse this any time from Finder → Get Info → Open With.

RunEvent::Opened
Single Tauri event covers both cold + warm. Cold: stored in Mutex state, read on mount. Warm: emitted directly to frontend.
No plugin needed
macOS single-instance behavior is native. tauri-plugin-single-instance only needed for Windows/Linux paths.
Multi-select
Selecting multiple .jsx files and hitting Enter opens them all. Each becomes a separate tab in the focused pane.
Drag to Dock
Dragging a .jsx onto the JSX Viewer dock icon while running also triggers RunEvent::Opened. Same warm path.
